🌑 NOCTURNE — Luxury E-Commerce Experience

Luxury Exists In The Dark.

Nocturne is a premium full-stack luxury e-commerce platform built using the MERN stack. The project focuses on creating a cinematic, immersive, and production-ready shopping experience while implementing real-world e-commerce architecture including authentication, product management, order management, image uploads, admin controls, and responsive UI design.
